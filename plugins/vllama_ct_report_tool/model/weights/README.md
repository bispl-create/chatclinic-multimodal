# MS-VLM CT Report Weights

Place the local model weights for `ms_vlm_ct_report_tool` in this directory.

Expected default filenames:

- `brainct_ckpt_epoch6.pth`: MS-VLM brain CT report-generation checkpoint
- `checkpoint0009.pth`: DINO/vision encoder checkpoint

Large weight files are intentionally ignored by git.

Download these files from the Google Drive artifact link shared outside the pull request.
